# Definition of Done

Một trang Admin hoàn thành khi:

- Đúng tài liệu module.
- Đúng design system.
- Dùng layout chung.
- Responsive cơ bản.
- Có dữ liệu mock.
- Có filter UI.
- Có nội dung chính.
- Có modal/drawer nếu cần.
- Có loading.
- Có empty.
- Có filter-empty.
- Có error.
- Có permission denied.
- Không lỗi console.
- Tailwind build thành công.
- Link điều hướng hoạt động.
