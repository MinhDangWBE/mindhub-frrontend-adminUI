# Tổng quan MindHub Admin FE

MindHub Admin là giao diện quản trị cho nền tảng bán khóa học trực tuyến.

Phạm vi GD1 gồm:

- Dashboard
- Người dùng
- Nâng cấp giảng viên
- Khóa học
- Kiểm duyệt khóa học
- Danh mục
- Đơn hàng
- Doanh thu
- Rút tiền
- Tài khoản nhận tiền
- Bình luận và đánh giá
- Báo cáo
- Banner
- FAQ
- Thông báo

Giai đoạn hiện tại chỉ xây dựng giao diện tĩnh.
