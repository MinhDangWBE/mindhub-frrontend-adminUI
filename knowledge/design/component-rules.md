# Quy tắc Component

## Card

- Nền trắng.
- Border mảnh.
- Radius 24px.
- Padding 20px.
- Shadow rất nhẹ.

## Button

### Primary

- Nền tối.
- Chữ trắng.
- Radius 18px.
- Cao khoảng 36–40px.

### Secondary

- Nền xám nhạt.
- Chữ tối.
- Radius 18px.

### Destructive

- Chỉ dùng đỏ cho lỗi, xóa, khóa hoặc từ chối.

## Input

- Nền xám nhạt.
- Radius 18px.
- Cỡ chữ 14px.
- Có focus ring.

## Badge

- Radius 18px.
- Cỡ chữ 12px.
- Luôn có text, không chỉ dựa vào màu.

## Table

- Header rõ ràng.
- Có hover row.
- Có empty state.
- Action ở cột cuối.
- Có overflow ngang trên màn hình nhỏ.
