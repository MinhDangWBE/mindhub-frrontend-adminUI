# Status Dictionary

## User

- active → Đang hoạt động
- inactive → Tạm ngưng
- locked → Bị khóa

## Course

- draft → Đang hoàn thiện
- pending_review → Chờ duyệt
- approved → Đã duyệt
- rejected → Bị từ chối
- published → Đang công khai
- hidden → Đã ẩn

## Order

- pending → Chờ thanh toán
- paid → Đã thanh toán
- cancelled → Đã hủy
- failed → Thất bại
- expired → Hết hạn

## Revenue

- pending → Chờ ghi nhận
- available → Khả dụng
- withdrawn → Đã rút
- cancelled → Đã hủy

## Withdrawal

- pending → Chờ xử lý
- approved → Đã duyệt
- rejected → Bị từ chối
- paid → Đã thanh toán
- cancelled → Đã hủy

## Payout account

- active → Đang hoạt động
- inactive → Đã tắt
- pending_verification → Chờ xác minh
- rejected → Bị từ chối

## Comment

- visible → Đang hiển thị
- hidden → Đã ẩn
- deleted → Đã xóa
